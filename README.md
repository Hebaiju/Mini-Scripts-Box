# 小脚本代码合集

日常使用的各类自动化小脚本，统一仓库管理。

## 项目列表

| 项目 | 说明 | 运行环境 |
|------|------|----------|
| [雨云自动注册](./雨云自动注册/) | Selenium 自动批量注册雨云账号，集成腾讯点选验证码识别 | Windows / Python |
| [润雨云容器多程序运行](./润雨云容器多程序运行/) | MCSM 容器内多 Java 服务管理脚本，支持面板停止信号、自启动 | Linux / MCSM |
| [樱花frp同步启动多个frpc](./樱花frp同步启动多个frpc/) | 同步启动多个 frpc 隧道的 shell 脚本 | Linux |

## 安全提醒

- 各子项目的敏感文件（账号密码、密钥等）已通过 `.gitignore` 排除，请勿将敏感数据提交到仓库
- 推送前请检查 `git diff --cached` 确认无敏感内容
